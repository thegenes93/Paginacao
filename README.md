Intruções para inicializar 
Necessita instalar o Node v10 
Mysql

1-abra o CMD na pasta do projeto
2-DIGITE      npm install
3-Para iniciar     npm start
4-projeto na localhost

Obs: O software carrega todas as tabelas ao iniciar só e necessario criar o banco de dados " teste ".
     Necessario configurar o arquivo .env com as configurações do banco de dados local
